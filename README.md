# Forum

A Basic Forum build with the Laravel and Vue js.

### Requirement

- PHP >= 7.1.3
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
- Ctype PHP Extension
- JSON PHP Extension
- MySQL Database
- Redis Storage
- Algolia Account for Index Search
- Google Recaptcha Account

### Installation

```bash
    $ composer install
    $ npm install
    $ php artisan migrate
```
